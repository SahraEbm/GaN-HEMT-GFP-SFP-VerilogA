# GaN-HEMT-GFP-SFP-VerilogA
GaN HEMT with Gate and Source Field Plates Compact Model (Verilog-A) Physics-based modular Verilog-A compact model for AlGaN/GaN HEMTs featuring both Gate Field Plate (GFP) and Source Field Plate (SFP). Developed in collaboration between Amirkabir University of Technology and EPFL. Suitable for power electronics and RF/microwave circuit design.
